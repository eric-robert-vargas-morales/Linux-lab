## Comandos Aprendidos

### Navegacion
- ‘pwd‘ - Print working directory
- ‘cd‘ - Change directory
- ‘ls‘ - List files
- ‘tree‘ - Visualizar estructura

### Ayuda
- ‘man‘ - Manual pages
- ‘--help‘ - Ayuda rapida

### Git Basico
- ‘git init‘ - Inicializar repositorio
- ‘git add‘ - Agregar al staging area
- ‘git commit‘ - Guardar cambios
- ‘git status‘ - Ver estado
- ‘git log‘ - Ver historial
